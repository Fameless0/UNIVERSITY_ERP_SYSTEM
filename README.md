# ERP-Management

data insertion in the json format

eg - 
{

  "password": "test1234",
  "role": "Student"

"Student"

"Faculty"

"Admin"

"Librarian"

"Accountant"

DB_HOST=   //hostname eg : localhost
DB_USER=   //user created in mariadb
DB_PASSWORD=  //mariadb's user password
DB_NAME=     //database name
